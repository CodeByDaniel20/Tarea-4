# Tarea-4

Sistema de gestión de clientes y reservas hecho con Python y Tkinter.

## Funcionalidades
- Registro de clientes con validaciones.
- Inicio de sesión de clientes registrados.
- Creación y cancelación de reservas.
- Gestión de servicios disponibles.
- Registro de eventos en archivo de log.

## Archivos principales
- `main.py`: interfaz principal y flujo de la aplicación.
- `cliente.py`: clase `Cliente`.
- `reserva.py`: clase `Reserva`.
- `servicios.py`: lista de servicios disponibles.
- `excepciones.py`: excepciones personalizadas.
- `logs.py`: configuración de logs.

## Requisitos
- Python 3.x
- Tkinter incluido con Python

## Ejecución
```bash
python main.py
```

## Uso
1. Registrar un cliente.
2. Iniciar sesión con ese usuario.
3. Crear una reserva desde la ventana principal.
4. Cancelar reservas si es necesario.
